# Googie the arti facial word speaker

### Install & run
open terminal
```
git clone https://github.com/streamgao/googie-hbs.git && cd googie-hbs
npm install
npm start
```
open browser:  http://localhost:3000


### Routes
#### default(client)
i.e. http://localhost:3000

#### admin
i.e. http://localhost:3000/admin
